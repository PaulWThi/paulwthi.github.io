Native macOS terminal multiplexer

Run code, worktrees, and Git context in one calm command deck.

aiya is a Ghostty-powered desktop terminal built for development workflows: split panes, project-aware navigation, Git diffs, notifications, and isolated worktree builds that can run side by side without stepping on each other.

Session note

Each worktree gets its own app name, bundle ID, and DerivedData path, so multiple local builds can stay open at the same time.

What’s in the app

Built for active development, not just terminal nostalgia.

Split-pane terminal rendering

Ghostty-backed panes with drag-and-drop layout control and room for parallel work.

Project-aware sidebar

Keep projects, worktrees, and top-level context visible without living in separate windows.

Git diff visibility

Track status, changes, and worktree state alongside the code you are editing.

Desktop notifications

Get alerted when something important happens instead of polling terminals by hand.

In-app observability

Structured logs, a debug console, and internal tooling for understanding what the app is doing.

Worktree-safe builds

Per-worktree product names, bundle IDs, and DerivedData keep parallel builds isolated.

Workflow

One app, three layers of context.

01

Run terminals side by side

Keep build output, editor-adjacent shell work, and Git commands visible at the same time.

02

Track the repo while you work

Use the sidebar, changes panel, and diff view to stay anchored in project state.

03

Keep worktrees isolated

Spin up multiple builds without name collisions, bundle ID conflicts, or shared DerivedData churn.
